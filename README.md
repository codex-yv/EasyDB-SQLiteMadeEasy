# EasyDB

**EasyDB** is a GUI-based tool to manage SQLite databases **without writing a single line of code**.  
Many times, working with SQLite can be frustrating when you have to repeatedly write code just to update, delete, or add data. EasyDB makes this process simple and visual.

---

## 🚀 Features

### 1. Convert CSV to SQLite
- CSV headings must be strings.
- Automatically checks each column and lists those with unique values (potential primary keys).

### 2. Convert SQLite to CSV
- Export your database tables into CSV format easily.

### 3. Edit SQLite
- **Tree view** of SQLite data for easy navigation.
- **Row-wise updates** without writing SQL queries.

### 4. Edit CSV
- Edit CSV data directly in the tool before converting or importing.

### 5. Create a SQLite Database
- Create a new database without any SQL code.
- Set column names and properties:
  - Text
  - Integer
  - Primary Key
  - etc.

### 6. Add Data to SQLite
- Add data one by one or in bulk.
- Supported formats:
  - List of tuples
  - List of dictionaries
  - CSV file
